One Dataset Shown In Four Ways

This dataset allows you to view residential and commercial permits issued by DPIE (Department of Permitting, Inspections, and Enforcement) since July 1, 2013.

There are four graphs showing different features.

According to Residential and Commercial Permits, the dataset has originally 44870 rows.

Due to API restrictions, it only allows 1000 limits, meaning the total number is 1000 instances.

Source code on Glitch and GitHub.
